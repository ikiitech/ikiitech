M. Rifki Sumarna - Kali ini saya akan membagikan tutorial, Cara membuat 2 adapter jaringan di Debian virtualbox. Simak tutorialnya berikut ini.

Pertama kita koneksikan Windows ke internet bisa melalui Wi-Fi atau LAN.

Lalu buka Virtualbox, buka pengaturan OS virtualnya, pilih Network. Arahkan adapter 1 ke Host-Only Adapter.

Geser ke adapter 2 kita arahkan ke Bridged Adapter dan pilih adapter jaringan yang digunakan di windows.

Jalankan debian virtualnya dengan klik Start.

Login menggunakan akun root supaya tidak usah pakai perintah su untuk masuk ke super user / root.

Kita konfigurasi pengaturan jaringannya di debian dengan perintah> nano /etc/network/interfaces

Tambahkan adapter jaringannya di bawah adapter enp0s3 / Host-only. Bedanya enp0s3 dengan enp0s8 adalah jenis konfigurasinya, enp0s3 untuk host-only berarti menggunakan static dan enp0s8 untuk internet berarti menggunakan dhcp. Setelah itu simpan dengan cara CTRL+X, lalu Y dan enter.

Setelah dikonfigurasi, restart service jaringannya dengan perintah> /etc/init.d/networking restart

Perhatikan IP disetiap interfaces, jika sudah ada berarti konfigurasi Jaringan host-only dan internetnya sudah berhasil.

Kita coba ping ke IP Google dengan perintah> ping 8.8.8.8

Coba lagi dengan perintah> ping google.com

Jika keduanya memiliki output ttl dan time berarti sudah berhasil terkoneksi ke internet.

Lanjut ke konfigurasi koneksi antara client dengan server, cari view network connections.

Pilih adapter Virtualbox host-only yang digunakan di virtualbox.

Lalu klik kanan adapter jaringannya, pilih properties.

Klik Internet protocol version 4 (TCP/IPv4), lalu pilih Properties.

Ubah ke static / manual lalu isi dengan ip client yang satu segmen dengan server.

Lalu uji coba ping dari client ke server.

Uji ping juga dari server ke client.

Jika suda ttl maka sudah berhasil.

Jika tidak bisa ping berarti Firewall di windows masih hidup, caranya cari firewall lalu klik open.

Pilih turn windows defender firewall on or off.

Pilih turn off windows defender firewall. Jika sudah klik OK.

Sampai di sini kita sudah berhasil membuat 2 adapter jaringan di virtualbox, bisa untuk jaringan lokal dan konek ke internet.
