M. Rifki Sumarna - Kali ini saya akan membagikan tutorial cara membuat web server di Debian menggunakan nginx. Simak tutorialnya berikut ini.

Pertama pastikan pengaturan Network di virtualbox mengarah ke Host Only agar bisa diakses oleh Windows yang berperan sebagai client.

Jalankan debian, lalu login menggunakan user root. Yang pertama kita konfigurasi IP menggunakan perintah nano /etc/network/interfaces. Ubah dari dhcp ke static seperti gambar di atas. Di sini kita gunakan Debian 10 sebagai server dan windows 11 sebagai client. Masukkan IP Address sesuai dengan keinginan kamu, setelah itu simpan dengan cara CTRL+X, lalu Y dan enter.

Kemudian kita konfigurasi IP di client Windows melalui control panel, caranya cari control panel di menu Search lalu enter, pilih Network and Internet, lalu pilih Network and Sharing Center, lalu pilih Change adapter setting di pojok kiri atas. Kemudian pilih adapter jaringan VirtualBox Host-Only yang digunakan di VirtualBox, klik kanan lalu pilih Properties, klik dua kali Internet Protocol Version 4 (TCP/IPv4), Pilih Use the following IP Address, misalkan IP Debian 40.40.40.1/24 maka di sini kita masukkan IP dari 40.40.40.2 sampai 40.40.40.254 BEBAS, subnet mask isi dengan 255.255.255.0 jika IP Debian menggunakan prefix /24, jika sudah klik OK.

Uji ping dari server ke client dan dari client ke server, pastikan reply.

Lanjut ke instalasi paket nginx. Di sini kita gunakan DVD 1 & 2 sebagai bahan instalasi. Ketik perintah apt-cdrom add.

Pilih menu Devices, lalu pilih Optical Drives, pilih debian DVD 2 atau jika belum ada maka pilih Choose a disk file, cari letak file iso DVD 2 berada. Jika sudah klik enter.

Jangan panik jika ada output Failed seperti pada gambar di atas yang dilingkari warna merah.

Ketik perintah apt-cdrom add lagi agar DVD 2 terbaca oleh sistem.

Setelah itu kita lanjut ke instalasi paket nginx, ketik perintah apt-get install nginx, enter lalu jika ada pilihan y/n maka ketik y.

Biasanya akan diminta untuk memasukkan DVD 1.

Caranya sama seperti saat memasukkan DVD 2.

Jika ada pop up seperti gambar di atas, maka pilih Force Unmount, lalu tekan enter.

Lalu akan diminta untuk memasukkan DVD 2 lagi. Tunggu sampai proses instalasi selesai.

Setelah berhasil terinstall biasanya service nginx akan berjalan secara otomatis. Untuk melihat sudah berjalan atau belum, ketik perintah /etc/init.d/nginx status lalu tekan enter. Jika berjalan maka output-nya Active (running) seperti gambar di atas.

Terakhir tinggal uji coba tes web server melalui browser seperti Google Chrome. Ketik IP server debian di kolom pencarian lalu enter dan lihat hasilnya.

Sampai di sini kita sudah berhasil membuat web server menggunakan nginx di sistem operasi debian linux.
